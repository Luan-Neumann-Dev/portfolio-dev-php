## 🎯 About

This project is a personal portfolio built with PHP to showcase projects and professional information in a simple and organized way.  

It uses a component-based structure with reusable sections, making the code easier to maintain and extend.

## ✨ Features

- 📱 **Responsive layout** - Built using Tailwind CSS via CDN
- 🧩 **Component-based structure** - Header, Hero, Projects, and Footer separated
- ⚡ **Dynamic rendering** - Projects are generated from a PHP array
- 🎨 **Dynamic tech badges** - Styles vary based on each technology

## 🛠️ Built With

- **PHP** - Handles structure and dynamic rendering
- **HTML5** - Page structure
- **CSS3 (Tailwind)** - Styling via CDN
- **JavaScript** - Minimal usage (Tailwind support)

## 🚀 Getting Started

### Prerequisites

- PHP installed (recommended PHP 7+)
- A modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio-dev-php.git

# Navigate to project directory
cd portfolio-dev-php

# Start local server
php -S localhost:8000

# Open in browser
http://localhost:8000
```

## 📁 Project Structure

```
portfolio-dev-php/
├── index.php              # Entry point
├── server.php             # Simple server config
├── components/
│   ├── header.php         # Navbar
│   ├── hero.php           # Introduction section
│   ├── projects.php       # Dynamic project list
│   └── footer.php         # Footer
├── img/                   # Images and assets
```
